# RunModeratorSim

Простая игра на андроид о сложной жизни модераторов speedrun.com.

**Правила просты**

- Раны с не правильным **IGT** должны быть **реджекнуты**
- Раны с не правильной игрой должны бить **реджекнуты**
- Раны **не** сталкера должны быть **реджекнуты**
